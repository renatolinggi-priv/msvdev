Integration in deine Anwendung

1. Als eingebettetes Bild

<img src="zielscheibe/zielscheibe_anzeigen.php?schuetze_id=123&serie_id=456" 
     alt="Zielscheibe" 
     style="max-width: 100%; height: auto;">

2. In PHP mit eigenen Daten

<?php
require_once 'zielscheibe/ZielscheibeGenerator.php';

$treffer = [
    ['schuss_nr' => 1, 'wert' => 10, 'x' => 14.47, 'y' => 5.08],
    ['schuss_nr' => 2, 'wert' => 9, 'x' => 28.9, 'y' => 46.77],
];

$generator = new ZielscheibeGenerator(800, 800);
$generator->generiereZielscheibe($treffer, 'meine_zielscheibe.png');
?>

3. Aus der Datenbank

⚠️ Wichtig: Passe die Tabellen- und Spaltennamen in zielscheibe_anzeigen.php an deine Datenbank an!

Nächste Schritte

  1. Prüfe die Zielscheibe oben - sehen die Treffer korrekt aus?
  2. Falls die Ringradien nicht stimmen, passe sie in der ZielscheibeGenerator.php an
  3. Integriere die Zielscheibe in deine Resultat-Seiten
  4. Passe zielscheibe_anzeigen.php an deine Datenbank-Struktur an
